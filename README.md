# Видеоконференция

Простое приложение для видеоконференций до 4 участников с демонстрацией экрана.

## Функции
- Создание и присоединение к комнатам
- Видеозвонки с аудио
- Включение/выключение вебкамеры
- Демонстрация экрана
- Ограничение до 4 участников

## Запуск локально
1. Установите зависимости: `npm install`
2. Запустите сервер: `npm start`
3. Откройте http://localhost:8080 в браузере

## Развертывание на Render.com
1. Создайте новый веб-сервис на Render.com
2. Подключите этот репозиторий
3. Установите команду запуска: `npm start`
4. Переменные окружения: PORT (автоматически устанавливается Render)

## Технологии
- Node.js
- Express
- Socket.IO
- WebRTC
